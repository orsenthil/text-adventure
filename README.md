## Text Adventure for Classics

I wanted to read Homer's The Odyssey . 

I thought, if we convert it into a text adventure with the instructions and passages directly from the text, then it
becomes a lot more enjoyable.

https://odyssey.learntosolveit.com/

Right now, it is only The Odyssey. but it could be could extend this to other classics.

- The Iliad (Homer)
- Gilgamesh
- Dante's Inferno
- Journey to the West
